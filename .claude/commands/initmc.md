# /initmc（已废弃 ⚠️）

> **此命令已废弃，请使用新的部署方式**

---

## ⚠️ 废弃说明

`/initmc` slash command 已在 v2.0.0 版本中废弃，原因：

1. **可靠性问题**: 依赖 AI 理解复杂的文件复制逻辑，容易出错
2. **维护困难**: 占位符替换逻辑复杂，难以调试
3. **用户体验差**: 执行时间长（3-15分钟），容易超时

---

## 🎯 新的部署方式

请使用全局命令 `initmc` 代替：

### 使用步骤

```bash
# 1. 在 MODSDK 项目根目录打开 cmd
cd <你的MODSDK项目目录>

# 2. 执行部署命令
initmc

# 3. 等待部署完成（约10-30秒）

# 4. 重启Claude Code（如果已打开）

# 5. 立即开始开发
/cc 创建一个玩家加入事件监听System
```

### 优势

- ✅ **可靠性高**: 使用 Node.js 脚本直接复制文件，不依赖 AI
- ✅ **速度快**: 10-30秒完成部署
- ✅ **自动验证**: 确保所有文件正确复制
- ✅ **定制化配置**: 自动替换项目路径等占位符
