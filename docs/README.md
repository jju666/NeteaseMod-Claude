## 🔄 文档更新策略

### 更新优先级

1. **Level 1 (超精简版)**: 保持精简，只更新核心规范和FAQ
2. **Level 2 (标准版)**: 定期同步核心机制和开发规范
3. **Level 3 (完整版)**: 添加新的详细代码示例

### 同步规则

```
CLAUDE-COMPACT.md (手动维护，保持精简)
    ↑ 定期同步核心内容
CLAUDE.md (主文档，定期更新)
    ↑ 同步核心章节
CLAUDE-FULL.md (归档，按需补充详细示例)
```

### 版本发布时的文档更新

- [ ] 更新所有文档的版本号（顶部声明）

- [ ] 更新 CLAUDE.md 的核心章节（如有重大变更）
