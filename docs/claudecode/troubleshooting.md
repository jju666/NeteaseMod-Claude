# 故障排除指南

这份文档提供了Claude Code安装和使用中常见问题的综合解决方案。

## 主要问题类别

**WSL安装问题**包括操作系统检测错误、Node找不到错误和nvm版本冲突。建议的修复涉及设置正确的npm配置或通过Linux包管理器安装Node。

**权限和身份验证**部分解决了重复的权限提示和登录失败。用户可以通过运行特定命令"重新启动您的shell，然后验证您的安装"来重置认证凭证。

**性能优化**建议包括使用`/compact`命令减少上下文大小和在WSL上将项目移到Linux文件系统以提高搜索性能。

**IDE集成**涉及WSL2防火墙配置、JetBrains终端中的ESC键绑定冲突以及"NetFirewallRule"设置说明。

**Markdown格式**问题涉及缺少代码块语言标签。解决方案包括请求Claude添加标签或使用后处理钩子实现自动格式化。

## 获取帮助

用户可以通过使用`/bug`命令、检查GitHub存储库、运行`/doctor`诊断或直接向Claude询问来获取额外支持。
