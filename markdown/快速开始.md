## ⚠️ 必知规范（防止90%错误）

开发前必须了解的CRITICAL规范：

### 1. 双端隔离原则

**禁止**: 跨端GetSystem（会返回None）
**应该**: 使用NotifyToClient/NotifyToServer通信

```python
# ❌ 错误
shop_client = self.GetSystem("ShopClientSystem")  # 返回None！

# ✅ 正确
self.NotifyToClient(player_id, "OpenShopUI", {"shop_id": 1})
```

### 2. System生命周期

**禁止**: 在__init__中调用GetComponent/GetEntity
**应该**: 在__init__中手动调用self.Create()

```python
def __init__(self, namespace, systemName):
    super(MySystem, self).__init__(namespace, systemName)
    self.comp = None
    self.Create()  # ⚠️ 必须手动调用！

def Create(self):
    self.comp = self.GetComponent(...)  # 在这里初始化
```

### 3. 模块导入规范

**应该**: 子目录使用完整路径导入

```python
# ✅ 正确
from myMod.config import CONFIG

# ❌ 错误
import config
```
